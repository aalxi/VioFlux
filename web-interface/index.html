Model Notes

Key constraints from the simulation model

VioE is critical: activity < 0.10 → hard zero (CPA diversion, no violacein)
VioC shunt: if VioC is weak relative to pathway median, flux leaks to chromoviridans (up to 35%)
Supply cap: limits max flux regardless of upstream activity
Burden penalty: over-expressing all genes strains the proteome — balanced is better than maximal
Imbalance penalty: large upstream activities relative to bottleneck cause metabolic buildup
